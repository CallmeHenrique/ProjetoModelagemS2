Recuperação de Senha

UMLUseCase
S² Home Broker :: Diagramas de Caso de Uso :: 1. Autenticação e Conta de Usuário :: Recuperação de Senha

Description

Fluxo Principal:

  1. Cliente acessa aba de perfil do usuário;

  2. Sistema exibe informações sobre o cliente;

  3. Cliente clica em "esqueci senha";

  4. Sistema exibe campo de endereço de e-mail;

  5. Cliente preenche o campo com seu e-mail;

  6. Sistema encaminha instruções de redefinição para o e-mail destinado;

  7. Cliente insere sua nova senha, vide instruções.

  8. Sistema exibe mensagem de senha redefinida.
