Personalizar Visualização Cotações

UMLUseCase
S² Home Broker :: Diagramas de Caso de Uso :: 5. Informações de Mercado :: Personalizar Visualização Cotações

Description

Fluxo Principal:

  1. Cliente acessa a plataforma.

  2. Cliente realiza o login no sistema.

  3. Cliente navega até a seção "Cotações".

  4. Cliente seleciona a opção "Personalizar Visualização".

  5. Cliente configura suas preferências de visualização, como escolha de ações, período de tempo, tipo de gráfico, etc.

  6. Sistema atualiza a visualização de cotações de acordo com as preferências configuradas pelo cliente.

  7. Cliente visualiza as cotações personalizadas.
