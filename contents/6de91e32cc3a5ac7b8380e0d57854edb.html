Ordem de Compra

UMLClass
S² Home Broker :: Diagrama de Classe :: Ordem de Compra

Description

A classe Ordem de Compra gerencia as ordens de compra de ações dentro do sistema, controlando o processo de aquisição de ativos no mercado. Ordem de Compra tem os atributos: quantidade, valor, status, gatilho. E os métodos da classe englobam: enviaOrdemDeCompra - Para enviar uma nova ordem de compra cancelaOdemDeCompra - Para cancelar uma ordem de compra modificaOrdemDeCompra - Para modificar uma ordem de compra statusOrdemDeCompra - Para visualizar o status da ordem de compra
