Description

A classe Cliente, referente às pessoas que se registram no sistema da SoyStock. O cliente tem os atributos: nome, email, documentos, cpf e senha E os métodos do cliente englobam: login - Fazer o login na plataforma cadastro - Cadastrar o cliente no sistema recuperarSenha - Recuperar a senha do cliente modificaCredenciais - Modificar as informações do cliente bloqueiaAcesso - Quando o cliente erra a senha após n tentativas
