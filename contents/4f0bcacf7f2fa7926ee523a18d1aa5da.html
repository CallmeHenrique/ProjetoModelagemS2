Carteira

UMLClass
S² Home Broker :: Diagrama de Classe :: Carteira

Description

A classe carteira se refere a carteira de ações do usuário no sistema Carteira tem os atributos: acoes, grupoDeAcoes, estiloDeCompra, estiloDeVenda E os métodos da classe englobam: visualizaAcoes - Para o usário conseguir visualizar a ação compraAcoes - Para o usuário comprar ação vendeAcoes - Para o usuário vender ação agrupaAcoes - Para o usuário ter um grupo de ações geraExtrato - Para o usuário ver os negócios realizados em um determinado período emitirAvisoDeNegociacao - Emitir um aviso quando o usuário estiver negociando no mercado fracionário
