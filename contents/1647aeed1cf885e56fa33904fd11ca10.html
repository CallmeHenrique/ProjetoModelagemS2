Description

A classe Ordem de Venda gerencia as ordens de venda de ações dentro do sistema, controlando o processo de desinvestimento de ativos no mercado. Ordem de Venda tem os atributos: quantidade, valor, status, gatilho. E os métodos da classe englobam: enviaOrdemDeVenda - Para enviar uma nova ordem de venda cancelaOdemDeVenda - Para cancelar uma ordem de venda modificaOrdemDeVenda - Para modificar uma ordem de venda statusOrdemDeVenda - Para visualizar o status da ordem de venda
