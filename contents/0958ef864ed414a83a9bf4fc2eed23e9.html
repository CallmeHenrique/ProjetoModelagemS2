Permitir OC OV

UMLUseCase
S² Home Broker :: Diagramas de Caso de Uso :: 4. Gestão de Carteira :: Permitir OC OV

Description

Fluxo Principal:

  1. Cliente acessa a plataforma.

  2. Cliente realiza o login no sistema.

  3. Cliente navega até a seção de compra e venda de ações.

  4. Cliente seleciona se deseja comprar ou vender uma ação.

  5. Cliente especifica os detalhes da ordem, como tipo de ordem, quantidade, preço, etc.

  6. Cliente confirma a ordem.

  7. Sistema verifica e processa a ordem, atualizando a carteira do cliente conforme a transação.
